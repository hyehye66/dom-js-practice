# Day 1 - DOM 선택과 텍스트 변경

## 실습 내용
- `document.querySelector()`로 DOM 요소 선택
- `.innerText`로 텍스트 읽고 쓰기

## 실험 결과
- 콘솔에서 직접 텍스트 변경 가능
- 버튼 클릭 시 텍스트가 JS로 바뀜

## 깨달은 점
- DOM은 HTML이 아니라 **브라우저 메모리 상의 구조**
- JS가 DOM을 조작하면서 화면이 바뀌는 원리 이해
